# Est-ce que Luna Park est fait pour moi ?

## Pour les no-codeurs

Luna Park se situe à peu près au même niveau de complexité que **Bubble** : vous devrez comprendre des concepts de **logique, de flux et de données**.
L'outil offre plus de **contrôle** et de **flexibilité** qu'un builder classique : ce n'est pas juste un assemblage de blocs.

## Pour les développeurs

La prise en main est **rapide**. La logique visuelle reste très proche de la structure du code que vous connaissez : vous allez retrouver toutes les fonctionnalités que vous attendez d’un **framework JS moderne** mais sans avoir à écrire de code. Vous pouvez créer des composants réutilisables, gérer l'état de votre application, et même créer des API backend.

## Pour les novices

Luna Park est parfait pour **apprendre le développement web**. Vous découvrirez les concepts fondamentaux d'une vraie application web (état, logique, données...) mais avec une **interface visuelle** qui rend l'apprentissage beaucoup moins intimidant que du code brut.

## Cas d'usage recommandés

Luna Park est idéal pour **créer des applications complexes et durables** ainsi que **des prototypes réutilisables**, vous permettant de concevoir un **MVP fonctionnel** et de **l'étendre sans recommencer de zéro**.
